Regex examples
r”\d{3}-\d{3}-\d{4}”  This line of code matches U.S. phone numbers in
the format 111-222-3333.

r”^-?\d*(\.\d+)?$”  This line of code matches any positive or 
negative number, with or without decimal places.

r”^/(.+)/([^/]+)/$” This line of code is often used to extract 
specific parts of URLs or file paths, such as the directory names or filenames.
